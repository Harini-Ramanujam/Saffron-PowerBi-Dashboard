# Saffron Production Analysis Dashboard

## 📌 Project Overview

The **Saffron Production Analysis Dashboard** is an interactive business intelligence project developed using Microsoft Power BI to analyze saffron production across different states, fiscal years, cultivation areas, and industrial purposes.

This dashboard provides meaningful insights into saffron production trends and helps users understand production patterns through interactive visualizations and filters.

---

##  Objectives

* Analyze saffron production across different states
* Study yearly production trends
* Compare saffron usage across industries
* Visualize cultivation area distribution
* Create an interactive and user-friendly analytical dashboard

---

## Dashboard Features

### ✅ KPI Cards

* Total Production
* Total Area
* Average Production
* Total States Covered

### ✅ Interactive Filters

* Fiscal Year
* Purpose
* State

### ✅ Visualizations Included

* Production Trend by Fiscal Year (Line Chart)
* Purpose-wise Production Analysis (Bar Chart)
* State-wise Production Distribution (Pie Chart)
* Area Distribution by State (Column Chart)
* Purpose Distribution Count (Donut Chart)

---

##  Tools & Technologies Used

* Microsoft Power BI
* Microsoft Excel / CSV Dataset
* DAX (Data Analysis Expressions)
* Power Query for Data Transformation
* Data Cleaning & Preprocessing
* Data Visualization Techniques

---

## 📈 Key Insights

* Jammu and Kashmir contribute the highest saffron production.
* Food industry usage dominates compared to cosmetics and pharmaceutical sectors.
* Production trends vary across fiscal years.
* Cultivation area distribution differs significantly between states.
